# AI Emotion & Interaction Platform

AI 기반 감정 분석 및 대화형 인터랙션 시스템을 실험하고 개발하는 프로젝트입니다.

---

## 🎯 Project Goal
사용자의 감정, 상황, 대화 흐름을 분석하여  
개인화된 응답과 콘텐츠를 제공하는 AI 기반 인터랙션 플랫폼을 구축하는 것이 목표입니다.

---

## 🚀 Key Features (Planned & In Progress)

- 감정 기반 음악 추천 시스템
- AI 대화형 커뮤니티 보조 기능
- Discord 봇 자동화 및 관리 시스템
- 사용자 맞춤형 응답 생성
- 실시간 데이터 저장 및 분석 (Firebase / Supabase)
- 웹 기반 인터페이스 (UI/UX)

---

## 🧠 AI & Integration Plan

- Kanana API 기반 자연어 인터랙션 실험
- 감정 분석 → 콘텐츠 추천 로직 설계
- 프롬프트 엔지니어링 기반 응답 최적화
- Discord / Web 연동 구조 구현

---

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask / Node.js 예정)
- Database: Firebase / Supabase
- AI APIs: Kanana API, OpenAI API (test)
- Integration: Discord Bot (discord.py), Webhook

---

## 📌 Project Status

현재 기능 설계 및 초기 개발 단계이며,  
핵심 기능들을 순차적으로 구현 및 테스트 중입니다.

## 🎯 Intended Use

이 프로젝트는 단순한 실험이 아니라  
실제 사용자 경험 기반 AI 서비스로 확장하는 것을 목표로 합니다.

- 감정 기반 추천 시스템 테스트
- 커뮤니티 자동화 기능 연구
- AI 인터랙션 UX 개선 실험
- 실제 서비스 적용 가능성 검증
